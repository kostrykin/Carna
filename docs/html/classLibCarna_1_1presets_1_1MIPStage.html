Detailed Description

Renders maximum intensity projections of volume geometries in the scene.

Basics

The MIPStage constructor takes a geometry type parameter:

mip = new presets::MIPStage( GEOMETRY_TYPE_VOLUMETRIC );
mip->colorMap.writeLinearSegment(
0, 1,
base::Color( 0, 0, 255, 255 ), base::Color( 255, 0, 0, 255 )
);
renderer->appendStage( mip );

The concept of geometry types is explained here.

Note
In the rendering process this stage will usually be inserted after such stages that render opaque geometry, like CuttingPlanesStage and OpaqueRenderingStage.

More complex color maps can easily be generated using the base::ColorMap::writeLinearSpline function:

std::vector< base::Color > colors =
{ base::Color( 0, 0, 255, 0 )
, base::Color( 0, 0, 255, 255 )
, base::Color( 0, 255, 255, 255 )
, base::Color( 255, 255, 0, 255 )
, base::Color( 255, 0, 0, 255 ) };
mip->colorMap.writeLinearSpline( colors );
exemplary rendering with a "jet"-like color map
